# ЯСНА - Астрология и натальные карты

Next.js приложение для консультаций по астрологии, натальным картам и эзотерике.

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env.local`:

```
JWT_SECRET=your-secret-key-here
N8N_WEBHOOK_URL=https://your-n8n-webhook-url
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Функционал

- Авторизация по телефону (мнимая - любой номер работает)
- Вход по телефону и паролю
- Сброс пароля
- Чат с поддержкой webhook через n8n
- История чатов (темы)

## База данных

Приложение использует MySQL базу данных. Таблицы создаются автоматически при первом запуске через Sequelize.
